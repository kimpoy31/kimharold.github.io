Event Attendance Monitoring System

This project is a lightweight attendance monitoring system developed for a one-time HR event with a large number of participants. The primary goal was fast deployment and reliable on-site usage rather than long-term scalability.

The system supports barcode scanning, QR code scanning, and manual attendee lookup for time-in and time-out tracking. It was designed to handle real-time event flow where speed and usability were critical, especially during peak entry and exit periods.

Attendance records can be reviewed and exported into a printable daily time record (DTR) format immediately after the event. Due to tight timelines, the system prioritizes simplicity and operational reliability over extensibility.

This project demonstrates my ability to quickly translate operational requirements into a working system, make practical technical trade-offs, and deliver a solution under real-world constraints.
